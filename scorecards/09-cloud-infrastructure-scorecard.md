# Cloud Infrastructure Scorecard

> Summarize the health, resilience, scalability, and operational maturity of the cloud infrastructure by consolidating the evidence produced during the Cloud Infrastructure Review.

---

# Purpose

The Cloud Infrastructure Scorecard provides an executive and technical assessment of the infrastructure platform supporting the software system.

Its objectives are to:

- Summarize the Cloud Infrastructure Review.
- Measure infrastructure maturity.
- Assess cloud-native adoption.
- Identify infrastructure risks.
- Prioritize platform improvements.
- Evaluate operational resilience.
- Support production readiness.

This scorecard does not perform a cloud infrastructure review.

It summarizes the engineering evidence produced by the Cloud Infrastructure Review.
