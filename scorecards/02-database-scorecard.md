# Database Scorecard

> Summarize the health, scalability, reliability, and maintainability of the database platform by consolidating the evidence produced during the Database Review.

---

# Purpose

The Database Scorecard provides an executive and technical assessment of the software's data platform.

Its objectives are to:

- Summarize the Database Review.
- Measure database health.
- Assess scalability and reliability.
- Identify data platform risks.
- Prioritize database improvements.
- Evaluate operational readiness.
- Support long-term data strategy.

This scorecard does not perform a database review.

It summarizes the engineering evidence produced by the Database Review.
