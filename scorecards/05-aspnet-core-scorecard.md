# ASP.NET Core Scorecard

> Summarize the health, maintainability, and engineering quality of the ASP.NET Core implementation by consolidating the evidence produced during the ASP.NET Core Review.

---

# Purpose

The ASP.NET Core Scorecard provides an executive and technical assessment of how effectively the application leverages the ASP.NET Core framework.

Its objectives are to:

- Summarize the ASP.NET Core Review.
- Measure implementation quality.
- Assess framework adoption.
- Identify architectural and framework risks.
- Prioritize framework-specific improvements.
- Evaluate maintainability.
- Support production readiness.

This scorecard does not perform an ASP.NET Core review.

It summarizes the engineering evidence produced by the ASP.NET Core Review.
