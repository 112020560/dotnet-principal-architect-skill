# Security Scorecard

> Summarize the security posture of a software system by consolidating the evidence produced during the Security Review.

---

# Purpose

The Security Scorecard provides an executive and technical assessment of the software system's security posture.

Its objectives are to:

- Summarize the Security Review.
- Measure the maturity of security controls.
- Assess cyber resilience.
- Identify security risks.
- Prioritize security improvements.
- Evaluate compliance with secure engineering practices.
- Support production readiness.

This scorecard does not perform a security review.

It summarizes the engineering evidence produced by the Security Review.
