# API Design Scorecard

> Summarize the quality, consistency, usability, and long-term evolvability of the API by consolidating the evidence produced during the API Design Review.

---

# Purpose

The API Design Scorecard provides an executive and technical assessment of the API contract and consumer experience.

Its objectives are to:

- Summarize the API Design Review.
- Measure API design maturity.
- Assess developer experience.
- Identify API design risks.
- Prioritize API improvements.
- Evaluate contract consistency.
- Support long-term API evolution.

This scorecard does not perform an API Design review.

It summarizes the engineering evidence produced by the API Design Review.
