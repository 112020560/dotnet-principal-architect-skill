# Principal .NET Architect Skill

This project contains an engineering knowledge base for reviewing
modern .NET applications.

The objective is not to generate code.

The objective is to evaluate architecture,
performance,
maintainability,
scalability,
security,
and operational readiness
using the criteria expected from a Principal Software Architect.

This skill supports:

- Claude Code
- Cursor
- OpenAI Codex
- OpenAI ChatGPT
- GitHub Copilot Chat
- Gemini CLI

It is designed around:

- .NET 8
- .NET 9
- .NET 10
- ASP.NET Core
- Hexagonal Architecture
- DDD
- CQRS
- Event Driven Architecture
- PostgreSQL
- SQL Server
- RabbitMQ
- MassTransit
- Docker
- Kubernetes
